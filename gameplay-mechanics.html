Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select All Mods -> Create Separator
  • Input Mechanics into the pop-up and select OK

Enabling More Features in JIP LN

  • Navigate to Fallout New Vegas/Data/nvse/plugins/ in and open jip_nvse.ini
  • Set bEnableFO3Repair to 1 (line 6)
    • Enables Fallout 3-style repair in which weapons can only be repaired up to a condition % equal to your repair skill
  • Set bNPCWeaponMods to 1 (line 72)
    • Allows NPCs to use weapon attachments

Enabling More Features in lStewieAl's Tweaks

  • Right-click on lStewieAl's Tweaks Custom INI mod (from Core - Bug Fixes/QOL) in the left pane of MO2 and select Open in Explorer
  • Open nvse/plugins/nvse_stewie_tweaks.ini in a text editor
  • Set bNoSkillTags to 1
    • Use control+f to easily find settings
  • Set iRemoveTags to 2
    • Additional changes can be made under the [No Skill Tags] section at the bottom of the INI
    • Hides the [Speech], [Barter], etc. number tags from dialogue options. For example, instead of [Speech 75], it will say just [Speech]
  • Set bHideEnemyMarkers to 1
    • Removes enemy markers from the compass like in Fallout 4 Survival
  • Set bNoGrabOwnedItems to 1
    • Stops the player from z-grabbing (not stealing) items owned by NPCs
  • Set bWeightlessWornPowerArmor to 1
    • Removes the weight of the power armor when you are wearing it like in Fallout 4
  • Set bSneakCriticalsOnlyMeleeWeapons to 1
    • Disables the ridiculously OP sneak attack automatic criticals with all weapons except melee/unarmed
  • Set bNoScaleNpcDamageThresholdByCondition to 1 and bNoScaleNpcDamageResistanceByCondition to 1
    • NPCs' armor DT/DR will no longer scale with condition (does not apply to the player)
  • Set bBarterCheckActorBuySellFlags to 1
    • Makes barter a lot more difficult by making the player only allowed to sell certain item types to certain vendors. If you enable it, install this mod which makes the items that certain vendors accept more logical

B42 Quickthrow

Installation instructions:

  • Download and install the B42 Quickthrow main file normally
  • Once the mod is installed, run FNVEdit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to Quickthrow.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Input 01008ED4 into the FormID search box in the top left and hit enter
  • In the right pane, right-click on MaleAdult01Default [VTYP:0000002D] next to VTCK - Voice and select Edit
  • Input VoiceDoNotRecordM [VTYP:00094EEC] into the pop-up box
    • ×
    • This tweak makes the "NPC" who essentially spawns in you to throw items silent, so he doesn't scream every time you throw something
  • Save and exit

- Adds a simple and modern quick throwing mechanic

Base Finish

Applying the Load Order

The final step is to apply the load order of installed plugins. The guide comes with a premade load order file, which ensures all mods are being loaded in correct order, with all conflicts accounted for.
  1. Download the VNV Load Order from here.
  2. In MO2, click the button at the top of MO2 and click Open Profile folder.
  3. Move loadorder.txt to the MO2 profile folder. You should get a popup message asking if you want to overwrite the existing file - click Yes.
  4. Press F5 while you are on MO2 without any menu open.

Your load order (found under the Plugins tab in the right pane of MO2) should match the loadorder.txt file you just downloaded. You can ignore any plugins from mods that you may have skipped, but the order the remaining plugins are in should match the file.

If that isn't the case, make sure to sort the load order by priority by clicking the Priority column header at the top.

Join our Discord server for support, if needed.

Running the Game

Due to the way Mod Organizer 2 works, running programs through it is a requirement in order to load mods. This is done in the top-right corner of the manager, where the program to run is chosen. Here, select New Vegas.

For the sake of convenience you may also create a shortcut (right below the Run button) so that you won't have to manually open MO2 each time before playing.

Q: Aren't you supposed to launch through NVSE?

A: The 4GB Patcher and the Epic Games Patcher both allow the game's main executable to automatically load the script extender.

Refer to the FAQ section for more information and solutions to possible problems you might encounter at this stage of the guide.


You have now completed the basic section of Viva New Vegas. If you don't want to mod your game further, you can now start playing. Have fun!

VNV Extended

Going beyond Vanilla

The next pages are part of the Viva New Vegas Extended portion of the guide, which is not required in order to play. This section includes:

  • UI and QoL improvements.
  • More gameplay features and improvements to get the game up to speed with more recent titles.
  • New and restored content, along with enhancements to the DLCs.
  • Overhauled balance and game mechanics.
  • Improvements to NPC faces, new animations, visual fixes, and overhauled lighting and weathers.

Note that VNV Extended is completely modular! You can skip any of its mods as long as you do the same for any other mod or patch tied to it.

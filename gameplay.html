Lone Star

A fixed, compatible, and slightly improved version of the JSawyer. Among the large number of (configurable) changes, here are the highlights:
  • Lower starting Health and Carry Weight.
  • Less loot.
  • Healing is harder.
  • Lackluster equipment is more viable.
  • Minor balance tweaks to perks in order to make them more interesting and fun to play.

Installation instructions

  • Main Files - Lone Star
    • You will be prompted with a FOMOD installer:
      1. Yes
      2. Install

JAM - Just Assorted Mods

Adds a highly configurable set of new modern features such as a quick loot menu, dynamic crosshair, sprinting, and more. Settings can be tweaked using the Mod Configuration Menu found in the in-game pause menu.

Installation instructions

  • Main Files - Just Assorted Mods
  • Viva New Vegas Resources
    • Main Files - JAM - VNV Configuration
  • JLM Grab Tweak
    • Main Files - JLM Grab Tweak
  • Clean Just Assorted Mods (JAM)
    • Main Files - Clean Just Assorted Mods

The Viva New Vegas preset for JAM only uses the following features:

  • Dynamic Crosshair - Modern crosshair that resizes in accordance to the weapon's spread.
  • Hit Marker - Crosshair markers signaling if you hit the target.
  • Hit Indicator - Arrows indicating the location of incoming damage.
  • Loot Menu - Fallout 4-style loot menu that opens whenever you look at a container.
  • Weapon Hweel - Quick-Use wheel menu.
  • Sprinting is disabled by default due to being replaced by Enhanced Movement.

Enhanced Movement

A configurable movement expansion - featuring movement bug fixes, 1st person diagonal movement, enhanced sprint, enhanced sneak, enhanced leaning, animated prone framework, new animations, and more.

Installation instructions

  • Main Files - Enhanced Movement
  • Main Files - Enhanced Movement INI
    1. Once the mod has been installed, double-click it in the left pane of MO2.
    2. In the INI Files tab, open NVSE/Plugins/DiaMoveNVSE.ini
    3. The recommended settings for VNV follow, but you're free to configure this differently:
      • Set iSprintSpeedBonus to 30 (line 85).
      • Set iAPCost to 10 (line 254).
      • Set bAgilityAffectsSpeed to 1 (line 297).
      • Set iAgilityMultiplier to 0.25 (line 302).
    4. Save your changes.

Gameplay

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Gameplay.
It is especially important to read the descriptions of these mods (You should still read the description of every mod), so you know what has changed and how to use the new features.

Just Assorted Mods

Installation instructions:

  • Main Files - Just Assorted Mods
  • Additionally install Clean Just Weapon Wheel
Highly configurable set of new modern features such as a quick loot menu, dynamic crosshair, sprinting, and more. Settings can be tweaked using the Mod Configuration Menu found in the in-game pause menu.

NPCs Sprint

Installation instructions:

  • Main Files - NPCs Sprint In Combat
Allows for melee-wielding enemies to use the new sprint feature from JAM.

Real Recoil

Installation instructions:

  • Main Files - Real Recoil v1.2.1
  • Additionally install Real Recoil Tweaks
Adds a visually smooth, truly dynamic, and realistic recoil system.

Follower Tweaks

Installation instructions:

  • Main Files - Follower Tweaks
Reworks Boone, ED-E, and Rexes' companion perks to be less intrusive/annoying.

Delay DLC Redux

Installation instructions:

  • Main Files - Delay DLC Redux
Prevents the game's DLC from spamming the new game startup messages and requires you be near a given DLC's starting location to receive its quest.

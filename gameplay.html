Gameplay

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select All Mods -> Create Separator
  2. Name the separator Gameplay

It is especially important to read the descriptions of these mods (you should still read the description of every mod), so you know what has changed and how to use the new features.

Enabling More Features in JIP LN
  1. Right-click on JIP LN NVSE mod (from the Utilities section) in the left pane of MO2 and select Open in Explorer
  2. Navigate to nvse/plugins/ and open jip_nvse.ini in a text editor
  3. Set bEnableFO3Repair to 1 (line 6)
    • Enables Fallout 3-style repair in which weapons can only be repaired up to a condition % equal to your repair skill
  4. Set bNPCWeaponMods to 1 (line 72)
    • Allows NPCs to use weapon attachments

Enabling More Features in lStewieAl's Tweaks
  1. Right-click on lStewieAl's Tweaks Custom INI mod (from the Bug Fixes/QOL section) in the left pane of MO2 and select Open in Explorer
  2. Navigate to nvse/plugins/ and open nvse_stewie_tweaks.ini in a text editor
  3. Use Ctrl+F to easily find and change the following options:
  4. Set bNoSkillTags to 1
    • Hides the [Speech], [Barter], etc. tags from dialogue skill checks
  5. Set bHideEnemyMarkers to 1
    • Removes enemy markers from the compass unless they are shooting
  6. Set bNoGrabOwnedItems to 1
    • Stops the player from z-grabbing (not stealing) items owned by NPCs

JSawyer Ultimate Edition

Installation instructions:

  • Main File - JSawyer Ultimate Edition 5.2.7
  • Optional File - JSawyer Ultimate Edition - GRA Merged 5.00

- An updated and fixed version of the gameplay re-balance mod made by the lead director of the game, Josh Sawyer

JSawyer Ultimate Custom INI

Installation instructions:

  • Download the mod using the download button pictured below
  • Finish the install normally through MO2

- Enables settings in the JSUE INI to make the game more difficult, mainly enabling the Survival Mode feature, which makes the player deal 1.5x more damage and enemies deal 2x more damage

With the previous two mods installed, it is highly recommended to play on High difficulty or lower.

Unfound Loot

Installation instructions:

  • Main File - Unfound Loot 1.0rc1
  • Optional File - Unfound Loot INI 1.0b11
Upon starting the installation of the Unfound Loot INI, MO2 will give a warning about the mod not having valid game data. You can ignore this warning and proceed with the installation. Once the mod has been installed, right-click on it in the left pane of MO2 and select Ignore missing data.

- Dynamically lowers the amount of loot in the game

Fallout 3 Economy

Installation instructions:

  • Main File - Fallout 3 Economy 2.0.0

- Overhauls item values to be similar to Fallout 3's much tougher economy

Follower Tweaks

Installation instructions:

  • Main File - Follower Tweaks 1.0

- Reworks Boone, Ed-E, and Rexes' companion perks to be less intrusive/annoying

Follower Formula Redone

Installation instructions:

  • Main File - Follower Formula Redone 1.0

- Changes the amount of followers you can have to your charisma skill/two rounded down, instead of always being able to have two

With this mod enabled, you will not be able to hire any followers if you have 1 Charisma.

GRA Unique Weapons Relocated

Installation instructions:

  • Main File - GUWR Balanced NVSE 1.50

- Places the unique weapons from the Gun Runners' Arsenal DLC throughout the wasteland, rather than them all just being sold by vendors

Just Loot Menu

Installation instructions:

  • Main File - Just Loot Menu 1.7a
  • Optional File - Container Animations 1.0

- Adds a highly-customizable Fallout 4/76-style quick loot menu

Just Dynamic Crosshair

Installation instructions:

  • Main File - Just Dynamic Crosshair 3.7

- Adds a highly-customizable dynamic crosshair that varies based on your skill, weapon spread, etc

Just Hit Marker

Installation instructions:

  • Main File - Just Hit Marker 3.3a

- Adds a highly-customizable FPS-style hit marker

The individual "Just" mods are used instead of Just Assorted Mods because I do not feel like the other mods fit the vanilla-plus style of the guide. If you want all the features of Just Assorted Mods however, you can install it instead of the previous three mods without any issues.

Marathoner

Installation instructions:

  • Main File - Marathoner v1.3

- Increases your movement speed 25% while not in combat

Real Recoil

Installation instructions:

  • Main File - Real Recoil v1.2.1

Real Recoil Custom INI

Installation instructions:

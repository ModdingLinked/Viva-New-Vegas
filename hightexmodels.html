Creating a New Profile:

In order to keep a full installation of just the Core module handy, we will create a copy of the profile:

  • Click the button on the top bar of MO2
  • Click on the Viva New Vegas - Core profile and select Copy
  • Input Viva New Vegas - Visuals for the new name then click OK
  • Select the Viva New Vegas - Visuals profile from the profile drop-down

Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Textures + Models into the pop-up and select OK

Pip-Boys to the People

Install normally


Manual FNVEdit editing required:

  • Run FNVEdit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to PipBoysToThePeople.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Expand the PipBoysToThePeople.esp header and then the Non-Player Character head within it
  • Right-click on the Doc Mitchell record in the left pane and select Remove
  • Select the Boomer Teacher record in the left pane
  • Right-click in the right pane and select Hide no conflict and empty rows
  • In the right pane, right-click on the empty box above NULL - Null Reference and select Remove
  • Save and exit

- Gives Pip-Boys to NPCs who came from Vaults
  NOTE: If you plan on using a Pip-Boy replacer, such as the wonderful Pip-Boy 2000 MK IV, NPCs will still have the vanilla Pip-Boy

Character Expansions Revised

Files to install:

  • Main File - Character Expansions Revised
  • Optional File - Character Expansions Revised - Unofficial Patch Plus

Special install instructions:

  • Click the button on the top bar of MO2 and select INI Editor
  • Navigate to the FalloutCustom.ini tab
  • Paste the following into the box under the [General] section:
    • bLoadFaceGenHeadEGTFiles=1

- Visual overhaul of characters’ faces that stays true to vanilla aesthetics

A Note About Character Overhauls:

Character Expansions Revised is my favorite face overhaul because it is the most vanilla-esque and minimal. If you want to use a more noticeable face overhaul,
the only real option is Qwinn's Refined FNV Redesigned 3. It is a cleaned up and fixed version of New Vegas Redesigned 3 by Dracomies that comes with
pre-built character face gen and many compatibility patches. See the mod page and the original mod page for screenshots/install instrucitions/etc.
The mod comes with patches for mods in previous sections that have already been merged, such as Legion AI Fix and Vendor Respawn Fix.
The FOMOD will not let you select the patches unless the files are present, so you will need to reactivate those mods temporarily.

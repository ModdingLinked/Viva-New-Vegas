Overhauls

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select All Mods -> Create Separator
  2. Name the separator Overhauls

This page contains mods that greatly overhaul the game by making it much more challenging (and more rewarding in turn) and adding new features. These mods are not recommended for a first playthrough of the game. It is especially important to read the descriptions of these mods (you should still read the description of every mod), so you know what has changed and how to use the new features.

Enabling More Features in JIP LN
  1. Right-click on JIP LN NVSE mod (from the Utilities section) in the left pane of MO2 and select Open in Explorer
  2. Navigate to nvse/plugins/ and open jip_nvse.ini in a text editor
  3. Set bEnableFO3Repair to 1 (line 6)
    • Enables Fallout 3-style repair in which weapons can only be repaired up to a condition % equal to your repair skill
  4. Set bNPCWeaponMods to 1 (line 68)
    • Allows NPCs to use weapon attachments

Enabling More Features in lStewieAl's Tweaks
  1. Right-click on lStewieAl's Tweaks Custom INI mod (from the Bug Fixes/QOL section) in the left pane of MO2 and select Open in Explorer
  2. Navigate to nvse/plugins/ and open nvse_stewie_tweaks.ini in a text editor
  3. Use Ctrl+F to easily find and change the following options:
  4. Set bNoSkillTags to 1
  5. Set iRemoveTags to 2
    • Changes skill checks to only show the skill type and not the required value (e.g. [Speech 50] becomes just [Speech])
  6. Set bHideEnemyMarkers to 1
    • Removes enemy markers from the compass unless they are shooting
  7. Set bNoGrabOwnedItems to 1
    • Stops the player from z-grabbing (not stealing) items owned by NPCs
  8. Set bBarterCheckActorBuySellFlags to 1
    • Makes vendors only buy/sell items related to their profession
  9. Set bLuckDoesntAffectGambling to 1
    • Prevents your Luck skill from having an effect on gambling, making it less easy to cheese casinos

Complex Vendors

Installation instructions:

  • Main File - Complex Vendors 1.0

- Made for use with the bBarterCheckActorBuySellFlags in lStewieAl's Tweaks, makes what certain vendors can/can't buy more sensible

JSawyer Ultimate Custom INI

Installation instructions:

  • Download the mod using the download button pictured below
  • Finish the install normally through MO2

- Enables settings in the JSUE INI to make the game more difficult, mainly enabling the Survival Mode feature, which makes the player deal 1.5x more damage and enemies deal 2x more damage

With this mod installed, it is highly recommended to play on High difficulty or lower.

Economy Overhaul

Installation instructions:

  • Main File - Economy Overhaul 3.0.2

- Overhauls the entire economy from the ground up, making buying/selling more difficult but more rewarding

Unfound Loot

Installation instructions:

  • Main File - Unfound Loot 1.0rc1

- Dynamically lowers the amount of loot in the game

Unfound Loot Custom INI

Installation instructions:

  • Download the mod using the download button pictured below
  • Finish the install normally through MO2

- Balances the settings in Unfound Loot to better fit the other mods in the guide like JSawyer Ultimate Edition and Economy Overhaul

Follower Formula Redone

Installation instructions:

  • Main File - Follower Formula Redone 1.1

- Changes the amount of followers you can have to your charisma skill/two rounded down, instead of always being able to have two

With this mod installed, you will not be able to hire any followers if you have 1 Charisma.

Better Character Creation

Installation instructions:

  • Main File - Better Character Creation 1.6

- Improves the character creation by speeding up the process, adding specialized gear based on your tag skills, and making Wild Wasteland an opt-in feature rather than a trait

Better Ghouls

Installation instructions:

  • Main File - Better Ghouls 1.1

- Removes ghouls "roar" animation that plays before they attack, meaning they now charge the player instantly - making them much deadlier

The Living Desert

Installation instructions:

  • Main File - The Living Desert - Main File 2.1

- Adds hundreds of new NPCs and scripted events that make the world feel much more alive and reactive

Because of all the new NPCs/scripts this mod adds it can be somewhat performance-intensive in some areas, especially on lower-end systems. If you have a weaker PC, you may want to skip this mod.

Mobile Truck Base Remastered

Installation instructions:

  • Main File - Mobile Truck Base Remastered 1.2

- Adds a vanilla-friendly new player home and fast-travel method

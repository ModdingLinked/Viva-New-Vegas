Introduction

Overview

Viva New Vegas (VNV) is a modding guide for Fallout: New Vegas that will carefully walk you through how to install all the mods you will need for a perfectly stable, smooth and enjoyable experience.

The guide is designed to be accessible for everyone, regardless of your modding experience. It is still incredibly important that you read each instruction carefully, even if you believe you are experienced enough to skip them. There are many small instructions that are vital for your game to function, and can be easily missed if you aren't careful. Although you don't need any modding experience to use this guide, it is assumed that you are relatively experienced with operating a computer in general.

Features

  • Stability: The guide includes all the essential stability enhancements.
  • Performance: Every tool, setting and mod in the guide has been carefully selected and play-tested to ensure the maximum possible performance.
  • Customizability: The guide is split into 2 sections; all the mods required for the game to work and perform better are contained in the Utilities section. The mods that can be considered subjective and that alter or add game content are in the Viva New Vegas Extended portion, which is fully modular.
  • Better Gameplay: The Extended portion of this guide (VNVE) includes mods that enhance gameplay whilst preserving the original game's artistic intent. One such mod is Lone Star, although it's not recommended for first-time players.

Required Software

  • An English copy of the Ultimate Edtion of the game from either Steam, GOG, or the Epic Games Store.
    • Only the English version of the game is supported for maximum compatibility - translations can conflict with mods and cause issues.
    • You may use a non-Ultimate Edition English copy of the game as long as you own all the DLC, including the pre-order bonuses.
    • The Polish-Czech-Russian (PCR) version of Fallout: New Vegas is fully supported as long as the game itself is in English.
    • The German release uses a "No Gore" version, making it incompatible with xNVSE. Only Steam copies of this version are supported as long as you use DepotDownloader to download the international EXE.
    • The Microsoft Store versions are not compatible with xNVSE, and thus not supported.
  • A 64-bit copy of Windows 10, or newer.
  • The latest supported drivers for your GPU, directly downloaded from the vendor (Nvidia, AMD, Intel or Qualcomm)
  • An archiving tool (NanaZip is recommended).

Required Hardware

As a general rule of thumb, if your computer was manufactured on or after 1st January, 2018. You will likely be able to play VNV/VNVE. This is due to a confluence of factors, namely:

  • The game over a decade ago, and was not particularly difficult to run even when it was new.
  • The game scales quite well with modern hardware and is much more CPU-intensive than GPU-intensive
  • This modding guide requires you to install mods that improve performance over the stock game.
For more details, please read the tables below.
  • The Minimum requirements table specifies only what you need to play VNV/VNVE at 1080p, 60FPS.
  • The Recommended requirements table specifies what you need for an enjoyable experience with the entire ModdingLinked catalogue.

Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Core - Finish Line into the pop-up and select OK

Core - Conflict Resolution

Files to install:

  • Main File - 1.8. Core - Conflict Resolution FOMOD

FOMOD Install Instructions:

  • Pick the All in One version if you installed all the mods from this step. If not, choose the Individual Patches option and choose all that apply to you

- Resolves conflicts between the mods from this module

Core - Load Order.txt

Files to install:

  • Main File - 1.9. Core - Load Order.txt

Special install instructions:

  • Extract loadorder.txt to Mod Organizer 2/profiles/Viva New Vegas - Core
  • Once the file is extracted, right-click in the left pane of MO2 and select All Mods -> Refresh

- Ensures your load order is correct for the Core module

  NOTE: This load order assumes you followed the entire guide with all the recommended options
  For example, if you use DarnUI or installed your own mods, those plugin-ins will need to be moved back into the correct spot after the new load order is applied

Using zMerge

  • Verify Core - Conflict Resolution.esp is in the lowest spot in your load order
  • Run zEdit through MO2
  • From the top drop-down menu, select zMerge, and from the bottom select Fallout NV
  • Click Start Session
  • Once zMerge is loaded, click the gear icon in the top left
  • In the Core tab, you can change the Application Theme to Night
  • In the Integration Settings tab, choose Mod Organizer 2 for Mod manager and Portable for the MO2 instance:
  • Hit the Detect button to automatically set your MO2 paths
    • If the application fails to find your instance or points to another game, just click the folder icon and manually set them
  • In the Merge Settings tab, make sure the Merge output path is pointed to your MO2 mods folder and both Disable plugins and Disable mods are un-checked
  • Click OK to exit the settings menu
  • Click the Create Merge button
  • In the Details tab, change the Name to Core - Merge and make sure the Merge Method is Merge down (improved)
  • In the Plugins tab, select the following:
    • CardMaster.esm
      ThrowableFixes.esp
      Melee Reach Revised.esp
      FIXforLegionArmors.esp
      FixHitESP.esp
      CassFix.esp
      Enslave Gannon fix.esp
      Caleb McCaffery NPC and Hat Bug Fixes.esp
      Nightkin Scripts Corrected.esp
      Rebreather Tweaked.esp
      Female White Glove Society Mask.esp
      Motion Sickness Reduction Mod.esp
      Eddie Hears An Explosion.esp
      BartonThornTrigger.esp
      Combat Music.esp
      Clean Character Creation.esp
      Glasses Fix.esp
      MuzzleFlash Control.esp
      Lonesome Road Roughin It Bedroll Kit Fix.esp
      Vendors Containers Respawn Fix - Simple Edition.esp
      Perks Day Tripper and Chemist Survivalist Style Fix.esp
      ConvenientCompanionInventory.esp
      JohnsonNashNCRAmmoFix.esp
      Void Minigame Bypass.esp
      Better Pickup Prompt.esp
      JIP Improved Recipe Menu.esp
      Quest Item Hider.esp
      Tutorial Killer.esp
      NoTrapsPopup.esp
      Simple DLC Delay.esp
      CardMaster DM.esp
      Vanilla UI Plus.esp (Only if you followed the VUI+ HUD Section)
      DarNifiedUINV.esp (Only if you followed the DarnUI HUD Section)
      Project FNVIcons - Challenge.esp
      Core - Conflict Resolution AIO.esp OR whatever patches you selected from the Conflict Resolution FOMOD
  • Click OK to exit the merge menu
  • Click the Build button in the top right to the right of the merge name
  • The merge should build successfully and you can exit out of the pop-up
  • Click the Create Merge button
  • In the Details tab, change the Name to JustModsMerged and make sure the Merge Method is Merge down (improved)
    • It is important that the merge .esp name is JustModsMerged.esp, or else the game will crash
  • In the Plugins tab, select the following:
    • JustDynamicCrosshair.esp
      JustVisualObjectives.esp
      JustHitMarker.esp
  • Click OK to exit the merge menu
  • Click the Build button in the top right to the right of the merge name
  • The merge should build successfully and you can exit out of the program
  • Right-click in the left pane of MO2 and select All Mods -> Refresh
  • The merges should appear in the left pane and activate them like normal
    • Move JustModsMerged.esp above Core - Merge.esp
  • Click the on the top bar of MO2 and select Merge Plugins Hide
  • Right-click on the Core - Merge listing, select Disable plugins, and then close
    • All the mods that were merged should now appear grayed-out in the left pane of MO2
    • Repeat for JustModsMerged
  • Use the same method to reactivate the mods when you need to rebuild the merge
    • When you reactivate the mods, they will reappear in your load order in alphabetical order. Reinstall the Load Order.txt files to get it back into order

Testing the Game

  • Run the game through MO2
  • Start up a new game and quickly run through the tutorial in Doc Mitchell's house
  • Once you have exited Doc's house, open up the MCM menu to verify all the entries display correctly
  • Spend at least 15-20 minutes in game doing whatever to make sure the game is running smoothly

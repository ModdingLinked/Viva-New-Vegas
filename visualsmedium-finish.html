Visuals - Load Order.txt

Installation instructions:

  • Extract the loadorder.txt from the archive to Mod Organizer 2/profiles/Viva New Vegas - Visuals and overwrite when prompted
  • Once the file is extracted, right-click in the left pane of MO2 and select All Mods -> Refresh

- Automatically ensures your load order is correct for the Visuals module
NOTE: This load order assumes you followed the entire guide with all the recommended options (though it will still work perfectly if you left out any mods). If you installed your own mods, those plugin-ins will be moved to the bottom of the load order and need to be manually moved back into the correct spot after the new load order is applied. If you do not know how to check where any mods you have added should go in the load order, it may be best to not add any mods not in the guide.

Testing the Game

  • Run the game through the New Vegas option in MO2
  • Start up a new game and quickly run through the tutorial in Doc Mitchell's house
  • Run through the tutorial missions with Sunny Smiles to verify everything is running smoothly and correctly

Tweaking the MCM

The Mod Configuration Menu can be accessed through the pause menu in-game. The settings are save-specific, so it is recommended to wait to edit the settings until you start the save you plan on actually playing on

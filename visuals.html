Simple Character Expansions

Overhauls the face of 170+ NPCs.

Installation instructions

  • Optional Files - Simple Character Expansions - YUPDate
    • You will be prompted with a FOMOD installer:
      1. Install

After installing Simple Character Expansions, you'll need to reinstall the Goodies mod in order to add an appropriate patch. To do so, follow these steps:

  1. Right-click Goodies under your list's Content separator, then click Reinstall Mod.
    • If you cleared your downloads folder, you'll have to redownload the mod.
  2. You will be prompted with a FOMOD installer:
    1. Better Brotherhood (only if you installed Better Brotherhood)
    2. Functional Post-Game Ending (only if you installed Functional Post-Game Ending)
    3. Simple Character Expansions
    4. Install
  3. Click Replace when prompted.

New Vegas Animation Overhaul - Guns

An almost complete reanimation of the game's weapons.

Installation instructions

  • HIPControl - Weapon Idle Position Adjuster - ESPless
    • Main Files - HIPControl
  • Main Files - New Vegas Animation overhaul - Guns

After installing NVAO, you'll need to reinstall the Iron Sights Aligned mod in order to patch the new animations. To do so, follow these steps:

  1. Right-click Iron Sights Aligned under your list's Bug Fixes separator, then click Reinstall Mod.
    • If you cleared your downloads folder, you'll have to redownload the mod.
  2. You will be prompted with a FOMOD installer:
    1. New Vegas Animation overhaul - Guns
    2. Yukichigai's Unofficial Patch
    3. Install
  3. Click Replace when prompted.
  4. Move Iron Sights Aligned under New Vegas Animation overhaul - Guns.
    • If MO2 doesn't allow you to move the mod, make sure that you are sorting the left pane by priority by clicking the column title.

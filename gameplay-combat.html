Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Combat + AI into the pop-up and select OK

BLEED

Files to install:

  • Optional File - BLEED NV

- Overhauls how damage and force is applied, making combat quicker and harder, and making previously unviable builds such as unarmed more viable

BLEED - No Always Knockdown Explosions

Files to install:

  • Main File - 3.1. Gameplay - BLEED No Always Knockdown Explosions

- Changes the "Always Knockdown" flag back to "Knockdown by Formula" for all explosions

Melee Cleave (a.k.a. Sweep)

Install normally

- Makes melee attacks able hit multiple enemies, similar to the Dark Souls series

Combat Enhancer

Install normally


Manual FNVEdit editing required:

  • Once the mod is installed, run FNVEdit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to CombatEnhancer.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Input 2A449 into the FormID search box in the top left and hit enter
  • Change the DATA - Value from 2700 to 3700
  • Save and exit
    • This edit fixes the typo in the maximum flee range, which was lower the the minimum

- Edits various game setting entries for NPCs making combat a bit more interesting

Enemy AI - Tactics - NPC Healing

Hide the following:

  • XFO - 9db - Enemies - NPC Healing - med.esp
  • XFO - 9dc - Enemies - NPC Healing - high.esp

Optional (Advanced) Tweaking Available:

  • Run GECK through MO2

  • Double-click FalloutNV.esm (An X should appear in the adjacent box)
  • Double-click XFO - 9db - Enemies - NPC Healing - low.esp and select Set as Active File then select OK
  • Wait for the GECK to finish loading
  • In the Object Window, select the All lising, then input aaForcehealDebug into the Filter box and hit enter
  • Double-click on the entry that shows up
  • Find the entry in the script the reads as follows:

      • if HealYes == 1
               pltarget.additem stimpak 1
               pltarget.equipitem stimpak 1
               pltarget.playidle 3rdpspecialidlestimpak
               set justHealed to 1
               if getrandompercent < 30
                       pltarget.forceflee
              endif
      endif

  • Replace it with the following (make sure the indentation stays the same):

      • if HealYes == 1
               pltarget.cios stimpak
               pltarget.playidle 3rdpspecialidlestimpak
               set justHealed to 1
      endif

  • Save and script and close out of the window
  • Save and exit from the GECK
    • This edit directly applies the healing spell to NPCs instead of adding a Stimpak to their inventory to prevent weapon bugs

- NPCs use more cover, smarter tactics, more creature differences, and NPC healing

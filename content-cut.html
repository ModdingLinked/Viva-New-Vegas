Creating a New Profile:

In order to keep a full installation of other modules handy, we will create a copy of the profile:

  • Click the button on the top bar of MO2
  • Click on the profile of the most recent module you installed and select Copy
  • Input Viva New Vegas - Content for the new name then click OK
  • Select the Viva New Vegas - Content profile from the profile drop-down

Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Content into the pop-up and select OK
  • Create another separator below that called Restored + Improved Content

Uncut Wasteland

Files to install normally:

  • Main File - Uncut Wasteland plus NPCs.
  • Optional File - Uncut Extra Collection

- Restores a huge amount of scenery and other random things that were patched out of the game post-release

Character Expansions Revised - Uncut Wasteland Patch

Files to install normally:

  • Optional File - Character Expansions Revised - UW

- Patch for Character Expansions Revised and Uncut Wasteland
NOTE: Only install if you installed Character Expansions Revised from the Visuals module

Cut Sewers Restored

The Moon Comes Over The Tower Restored

The Living Desert

Install normally

Mojave Raiders - The Living Desert Patch

Files to install normally:

  • Optional File - Mojave Raiders - The Living Desert Patch

Beyond the Beef Tweaks

Install normally

- Improves options and makes the conclusion of the Beyond the Beef quest more fulfilling

Vault 22 Flora Overhaul

Install normally

- Modifies Vault 22 with more flora, ivy, bushes, grasses, etc.
NOTE: This mod can be very heavy on performance due to it not having roombounds/portals and can be skipped without any issues

Vault 22 Flora Overhaul ESP Replacer

Install normally

- Removes new creatures, weapons, poison effects, crafting recipes, etc. (really any out of place changes), along with a few other misc tweaks

Lightweight Strip Overhaul

Files to install normally:

  • Main File - Lightweight Strip Overhaul - No Walls
    • The No Walls version is used instead of the regular version because of a couple bugs. The walls are still the junk variants from the outside and the walls used don't have LOD, so they pop in every time you enter the furthest cell in the strip. The regular version can be used without issues if that does not bother you

- Cleans up debris/litter in the Strip and it's casinos

The Mogul Mausoleum - A Lucky 38 Overhaul

Install normally

- An overhaul of the Lucky 38 penthouse that makes it feel less bland and visually uninteresting

The Mogul Mausoleum ESP Replacer

Install normally

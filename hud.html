User Interface

In this step, we will install various mods focused on interface visuals and its usability

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create separator.
  2. Name the separator User Interface.

The Mod Configuration Menu

Installation instructions:

  • Main Files - The Mod Configuration Menu
  • Update File - MCM BugFix 2
Allows select mods to be configured via a single in-game menu.

Vanilla UI Plus

Installation instructions:

  • Download the mod and install manually through MO2.
    • You will be prompted with a FOMOD when installing in which you can tweak the settings to your personal preference
Improves the user interface without compromising the original style.

Vanilla UI Plus - Dialog Background Add-on

Installation instructions:

  • Main Files - Vanilla UI Plus Dialog Background Addon
Adds back the dialog background that was removed in recent Vanilla UI+ versions.

Clean Vanilla HUD

Installation instructions:

  • Download the mod and install through MO2.
    • When prompted with a FOMOD when installing, click Install
  • Clean Companion Wheel
    • Main Files - Clean Companion Wheel 256x256 Edition
HD remake of the vanilla interface.

High Res Local Maps

Installation instructions:

  • Main Files - High Res Local Maps
Increases the resolution of local maps to a configurable target resolution.

High Resolution Screens

Installation instructions:

  • Main Files - High Resolution Screens
Increases the render resolution of in-game screens (Pip-Boy, terminals, character creation menu) to match game's screen resolution.

Faster Main Menu

Installation instructions:

  • Main Files - Faster Start Menu
Makes the game's main menu load faster.

Menu Search

Installation instructions:

  • Main Files - lStewieAl - Menu Search
Add Ctrl-F to search in PipBoy, Container, Barter, Recipe, Levelup Perks and Save/Load menus.

JIP Improved Recipe Menu ESPless

Installation instructions:

  • Main Files - JIP Improved Recipe Menu ESPless
Makes the crafting menu easier, more efficient and less tedious to use.

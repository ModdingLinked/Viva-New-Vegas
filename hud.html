User Interface

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator User Interface.

User Interface Organizer

Installation instructions:

  • Main Files - UIO - User Interface Organizer
Automatically ensures that multiple HUD extensions work together.

The Mod Configuration Menu

Installation instructions:

  • Main Files - The Mod Configuration Menu
  • Update File - MCM BugFix 2 (select the name that matches the file on Nexus when the installation prompt comes up)
Allows select mods to be configured via a single in-game menu.

Vanilla UI Plus

Installation instructions:

  • Download the mod and install manually through MO2
  • You will be prompted with a FOMOD when installing in which you can tweak the settings to your personal preference.
  • Additionally install Clean Vanilla HUD (just select Install in the FOMOD).
Improves the user interface without compromising the original style.
If you do not like the look of VUI+, you can instead try DarnUI. Installation instructions can be found in the FAQ.

Consistent Pip-Boy Icons

Installation instructions:

  • Main Files - 1. Consistent Pip-boy Icons
  • Main Files - 2. Consistent Addon Icons (select the name that matches the file on Nexus when the installation prompt comes up then click Install in the FOMOD)
  • Additionally install Clean Companion Wheel 256x256
Thousands of Pip-Boy icons re-made from scratch in HD along with many bug fixes.

High Res Local Maps

Installation instructions:

  • Main Files - High Res Local Maps
Increases the resolution of local maps to a configurable target resolution.

High Resolution Screens

Installation instructions:

  • Main Files - High Resolution Screens
Increases the render resolution of in-game screens (Pipboy, terminals, character creation menu) to match game's screen resolution.

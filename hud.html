Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator HUD + UI

Vanilla HUD Cleaned

Installation instructions:

  1. Download the Main File - Vanilla Hud Cleaned 0.381 and begin the installation normally through MO2
  2. You will be prompted with a FOMOD installer, in which you should select everything EXCEPT Clean Fonts for Darn then select Install

- Hand-made up-scaled/cleaned recreations of many vanilla HUD elements

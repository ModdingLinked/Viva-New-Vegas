Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create separator.
  2. Name the separator Content.

FPGE - Functional Post Game Ending

Adds a fully functional post-ending world with consequences and a vastly changed world.

Installation instructions

  • Main Files - Functional Post Game Ending
    • You will be prompted with a FOMOD installer:
      1. YUP Patch
      2. Install

You can safely ignore the Error failed to restore load order... message.

The Living Desert

Better Brotherhood

Tweaks to make the Mojave Brotherhood a better faction, including more sources of fame, fixed ending slide conditions, additional repair services, improved diversified leveled lists, and a new weapon.

Installation instructions

  • Main Files - Better Brotherhood
  • Viva New Vegas Resources
    • Miscellaneous Files - Better Brotherhood - YUPDate Patch
  • Optional Files - FPGE Patch (only if you installed Functional Post Game Ending)

Goodies

Installation instructions

      1. Better Brotherhood (only if you installed Better Brotherhood)

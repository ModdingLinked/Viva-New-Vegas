Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select All Mods -> Create Separator
  • Input Environment + Lighting into the pop-up and select OK

A Little More Lamplight

Install normally

- Fixes/revamps many of the street lights in Outer Vegas, Camp McCarran, Camp Golf, and the NCRCF, as well as adding in some new ones

Lucky 38 E3 Lights Restored

Install normally

- Restores the Lucky 38's lights that were shown at E3, but cut because of console limitations

Burning Campfire Redone

Install normally

- Fires actually light when you activate them

Burning Campfire Redone 2

Realistic Wasteland Lighting Redesigned

Install normally

- Improved version of the original RWL with more and better weathers, restored region-specific weathers, overhauled interior lighting, and many bug fixes. See here for screenshots

A Note About Weather/Lighting Mods:

If you do not like RWL, the two best performance-friendly alternatives are Clarity and Yellow Goodbye. Clarity is a simple tint remover and Yellow Goodbye is based off of Clarity, but still retains a little bit of the vanilla tint. Neither should have any bugs or performance loss.

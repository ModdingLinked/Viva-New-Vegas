Bug Fixes

Creating a Separator in MO2.

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Bug Fixes.

Yukichigai Unofficial Patch

Installation instructions:

  • Main Files - YUP - Base Game and All DLC
Fixes thousands of bugs with both the base game and DLCs.

Unofficial Patch NVSE Plus

Installation instructions:

  • Main Files - Unofficial Patch NVSE Plus
Fixes many bugs like YUP, but takes advantage of NVSE and JIP LN.

Improved AI (Navmesh Overhaul Mod)

Installation instructions:

  • Main Files - Navmesh Overhaul
  • Optional Files - Navmesh Overhaul YUP Patch
This mod fixes and improves vanilla game pathfinding errors and will provide a better and more challenging experience with the AI.

lStewieAl's Tweaks and Engine Fixes

Installation instructions:

  • Main Files - Stewie Tweaks
Collection of many configurable quality of life improvements and bug fixes. Settings can be further enabled/disabled/tweaked in the new Tweaks section added to the main menu.

Stewie Tweaks Essentials INI

Installation instructions:

  • Main Files - Stewie Tweaks Essentials INI
Preconfigured collection of tweaks, enabling basic quality of life feature, bugfixes and performance patches.

Combat Lag Fix (NVSE)

Installation instructions:

  • Main Files - Fast Weapon Lag Fix
Improves framerate in combat by fixing an engine bug that lowered framerate while attacking an enemy with a health-bar visible.

ISControl Enabler

Installation instructions:

  • Main Files - ISControl
Dynamically adds ISControl node system animation support and easy per-weapon sights adjustment.

Iron Sights Aligned

Installation instructions:

  • Main Files - Iron Sights Aligned
  • You will be prompted with a FOMOD installer, here select Yukichigai's Unofficial Patch then Install
Fixes all the iron sights on vanilla weapon meshes using ISControl Enabler.

New Vegas Mesh Improvement Mod

Installation instructions:

  • Main Files - New Vegas Mesh Improvement Mod
Optimizations and fixes for over one thousand meshes throughout the base game and DLCs.

Improved Lighting Shaders

Installation instructions:

  • Main Files - Improved Lighting Shaders
Fixes engine-level lighting bugs and allows for 4x the normal amount of active lights.

Climate Control NVSE

Installation instructions:

  • Main Files - Climate Control NVSE
Fixes bugs and adds utilities for weather mods.

Misc Audio Tweaks and Fixes

Installation instructions:

  • Main Files - Assorted Voice Popping Fixes
  • Main Files - Elijah Missing Distortion Fix
A collection of tweaks for certain audio files to fix popping issues and other annoyances.

ActorCause Save Bloat Fix

Installation instructions:

  • Main Files - ActorCause Save Bloat Fix
Fixes a save bloating issue related to actors detecting the player, improving save/load speed and decreasing save size 15-30% depending on playthrough length.

Strip Performance Fix

Installation instructions:

  • Main Files - Strip Performance Fix
Drastically improves performance on The Strip by getting rid of water LOD.

Fog-based Object Culling

Installation instructions:

  • Main Files - Fog-based Object Culling
Dynamically adjusts draw distance in accordance with the fog to improve performance with little to no visual difference.

Exterior Emittance Fix - NVSE

Installation instructions:

  • Main Files - Exterior Emittance Fix
Fixes incorrect exterior emittance handling which leads to particles being black.

MoonlightNVSE

Installation instructions:

  • Main Files - MoonlightNVSE
Fixes moonlight by making the moon the light caster instead of the sun.

Pip-Boy Shading Fix NVSE

Installation instructions:

  • Main Files - Pip-Boy Shading Fix
Fixes Pip-Boy's shading, making it not ignore lights around it.

Depth of Field Fix - NVSE

Installation instructions:

  • Main Files - Depth of Field Fix
Fixes depth of field effects bleeding over onto player's viewmodel.

Muzzle Flash Light Fix - NVSE

Installation instructions:

  • Main Files - Muzzle Flash Light Fix
Fixes muzzle flash lights not being culled after shot, leading to game still using them for illumination.

Strip Street Light Corrections

Installation instructions:

  • Main Files - Strip Street Light Corrections
Corrects several minor issues with the Strip's street lights.

Crafting Consistency Fix

Installation instructions:

  • Main Files - Crafting Consistency Fix
Fixes the inconsistent capitalization of workbench tab names and fixes the HH leather armors not being in the correct tab.

ExRB - Extended Roombounds

Installation instructions:

  • Main Files - ExRB
  • You will be prompted with a FOMOD installer, here select Full, then Cut Sewers and Install
  • ExRB - YUP Patch
Adds and tweaks roombounds in some interiors possibly improving performance.

Strip Lights Region Fix

Installation instructions:

  • Main Files - Strip Lights Region Fix
Fixes a vanilla issue about the Strip lights not showing in certain parts of the map.

PipBoyOn Node Fixes

Installation instructions:

  • Main Files - PipBoyOn Node Fixes NV
Fixes multiple pipboy:on nodes in the armor meshes.

Consistent Spread - Firearms Accuracy and Wobble Fix

Installation instructions:

  • Main Files - Consistent Spreadv1.1
Fixes inconsistent accuracy caused by vanilla weapon wobble and bugged character spread setting.

Ammo Script Fixes

Installation instructions:

  • Main Files - Ammo Script Fixes
Fixes several problems at the core level with how ammo scripts and effects work, plus some tweaks for consistency and fun.

Performance and Stability Guide

In-depth guide about how to fix the game's lackluster performance, display issues and alt-tabbing. Covers topics such as DXVK, framerate limiting, display modes, lag and HDR.

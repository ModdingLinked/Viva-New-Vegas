Bug Fixes

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator Bug Fixes

Yukichigai Unofficial Patch

Installation instructions:

  • Main File - YUP - Base Game and All DLC

- Fixes thousands of bugs with both the base game and DLCs

Unofficial Patch NVSE Plus

Installation instructions:

  • Main File - Unofficial Patch NVSE Plus

- Fixes many bugs like YUP, but takes advantage of NVSE and JIP LN

lStewieAl's Tweaks and Engine Fixes

Installation instructions:

  • Main File - Stewie Tweaks
  • Additionally install lStewieAl's Tweaks Custom INI (Install manually in MO2)

- Collection of many configurable quality of life improvements/bug fixes. The INI file provided by the guide enables all of the bug fixes and many of the essential quality of life improvements. Settings can be further enabled/disabled/tweaked in the new Tweaks section added to the main menu

ISControl Enabler

Installation instructions:

  • Main File - ISControl ESPless FULL

Vanilla Iron Sights Realligned

Installation instructions:

  • Main File - Vanilla Iron Sights Realligned

New Vegas Mesh Improvement Mod

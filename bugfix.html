Bug Fixes

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create separator.
  2. Name the separator Bug Fixes.

YUPDate - YUP Tweaks and Fixes

Various tweaks and enhancements for YUP utilizing the full suite of xNVSE script extenders, including configurability of various balance changes.

Installation instructions

  • Main Files - YUPDate
  • Main Files - YUPDate Config

After installing YUPDate, you must hide YUP - NPC Fixes (Base Game + DLC).esp contained in YUP!

  1. Open Yukichigai Unofficial Patch - YUP in Mod Organizer 2 by double-clicking it on the left pane.
  2. Go to the Filetree tab.
  3. Right-click YUP - NPC Fixes (Base Game + DLC).esp and click Hide.

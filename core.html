The Goal of the Core Module:

The goal of this module is to set up all the essential modding tools for FNV as well as some essential bug fixes/quality of life improvements that give a very similar feel to vanilla, just without all the bugs and oversights. It is intended for anyone with any level of modding experience to be able follow it easily. It is all also fully modular, meaning you can leave out any mods you don't like (the same goes for the other modules as well). All other changes such as graphic overhauls, gameplay mods, and new quests/weapons that you may be accustomed to in fully modded setups are covered in the other 3 modules.

Requirements:

  • An English copy of FNV with all the DLC from either Steam or GOG (Instructions for properly installing it are in the next step)
    • Only the English versions of FNV/FO3 are supported. If your game is not English, follow these steps to switch it
    • IMPORTANT: The Bethesda.net version does not work with the 4gb patcher, NVSE, or any of its plugins. It can not be used with this guide
  • Windows 7 or higher (64bit)
  • A text editor (I recommend Notepad++, but default Windows notepad will work)
  • An archiving tool (I recommend 7-Zip)
  • Latest GPU drivers (Either NVIDIA or AMD)
  • At least 15 gigabytes of free drive space for just the Core module, and 23 gigabytes for the entire guide
  • NOT compatible with a Tale of Two Wastelands; if you want to use TTW, use my TTW Guide
  • A Nexus Mods account
